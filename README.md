# Doha Education Hub

This is the monorepo for the Doha Education Hub MVP.

## Structure
- `backend/` — FastAPI backend (Python)
- `frontend/` — Next.js frontend (TypeScript)

## Quick Start
1. See each subfolder for setup instructions.
2. Backend: Python 3.11+, FastAPI, PostgreSQL
3. Frontend: Node.js 18+, Next.js 14+, Tailwind CSS

See the sprint backlog for week-by-week implementation steps.
