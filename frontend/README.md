# Frontend (Next.js)

## Setup
- Node.js 18+
- Yarn or npm

## Install dependencies
```bash
npm install
```

## Run the dev server
```bash
npm run dev
```

## Project structure (to be created):
- `pages/` — App routes
- `components/` — UI components
- `lib/` — API clients, utils
- `styles/` — Tailwind CSS
